- `tech-stack.md` — detected framework/library versions (Next.js, React,

## When to use this skill

- A user asks you to add a new page/component/API route → read
  `code-patterns.md` first to match the established pattern.
- A user asks what version of X the project uses → read `tech-stack.md`
  rather than inferring from imports.
- You're about to suggest a library or syntax and need to check it against
  what's already installed.
