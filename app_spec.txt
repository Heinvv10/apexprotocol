  <overview>
    Apex is a white-label Generative Engine Optimization (GEO) / Answer Engine Optimization (AEO) platform that helps brands capture visibility across AI-powered search engines like ChatGPT, Claude, Gemini, Perplexity, Grok, and DeepSeek.

    The platform provides:
    1. MONITOR - Track brand mentions across 7+ AI platforms in real-time
    2. CREATE - Generate AI-optimized content using brand voice and data
    3. AUDIT - Comprehensive technical site analysis for AI visibility
    4. SMART RECOMMENDATIONS ENGINE - Auto-generated, prioritized actionable recommendations (Core differentiator)

    Target Market: South African businesses initially, expanding to African markets
    Key Differentiator: Dashboard-first UI with Smart Recommendations Engine (not chat-based)
    Pricing Strategy: PPP-adjusted pricing (30-50% below US competitors)
  </overview>

  <core_features>
    <feature name="Smart Recommendations Engine" priority="critical">
      <description>
        Auto-generated, prioritized recommendations based on continuous analysis.
        This is the CORE DIFFERENTIATOR - not chat-based like competitors.
      </description>
      <sub_features>
        - Content Gap Analyzer: Identifies missing content topics where competitors rank
        - Schema Auditor: Validates and recommends schema.org improvements
        - Multi-Language Planner: Prioritizes language translations (focus: African languages)
        - Voice Search Optimizer: Optimizes content for voice/conversational queries
        - Priority Calculator: Scores recommendations by impact, effort, and confidence
      </sub_features>
      <priority_algorithm>
        Priority Score = (Impact * 0.4) + (Confidence * 0.3) + (Effort_Inverse * 0.2) + (Recency * 0.1)
        - Impact: 0-100 based on predicted visibility improvement
        - Confidence: 0-100 based on data quality and sample size
        - Effort_Inverse: 100 - (effort_hours * 5), capped at 0-100
        - Recency: Higher score for newly identified opportunities
      </priority_algorithm>
      <components>
        - RecommendationCard: Expandable card with problem/solution/impact
        - RecommendationList: Filterable, sortable list with board/calendar views
        - RecommendationDetail: Full page with implementation guide
        - QuickActions: One-click export to Jira/Trello/Asana/Slack
      </components>
    </feature>

    <feature name="GEO Score Dashboard" priority="critical">

        Technical Score (30%):
        - Schema implementation quality (40%)

        - Mobile responsiveness (10%)

        Content Score (35%):

        AEO Score (35%):
        - AI platform mention frequency (40%)
