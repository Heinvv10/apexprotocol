## Technical Documentation Findings (docs.searchable.com)

*Added: December 9, 2025 via Playwright documentation exploration*

### Scoring Methodology - Complete Breakdown

Searchable uses a **weighted composite scoring system** with three main categories:

#### Overall Health Score Formula
```
Overall Health = (Technical × 0.30) + (Content × 0.35) + (AEO × 0.35)
```

#### Technical Score Components (30% of overall)

| Component | Weight | Description |
|-----------|--------|-------------|
| Page Speed | 30% | Load times and performance metrics |
| Mobile Optimization | 25% | Mobile-friendliness and responsive design |
| Core Web Vitals | 20% | LCP, FID, CLS metrics |
| Technical SEO | 15% | Meta tags, canonical URLs, robots.txt |
| Crawlability | 10% | Site structure and indexability |

#### Content Score Components (35% of overall)

| Component | Weight | Description |
|-----------|--------|-------------|
| Content Quality | 30% | Depth, accuracy, freshness |
| Keyword Optimization | 25% | Target keyword usage and placement |
| Content Structure | 20% | Headings, formatting, readability |
| Internal Linking | 15% | Site navigation and link equity |
| Meta Information | 10% | Titles, descriptions, OG tags |

#### AEO Score Components (35% of overall)

| Component | Weight | Description |
|-----------|--------|-------------|
| Structured Data | 25% | Schema markup quality and coverage |
| Answer Format | 25% | Content formatted for AI extraction |
| Citation Worthiness | 20% | Authority signals that encourage AI citation |
| Semantic Richness | 20% | Entity relationships and context |
| AI Accessibility | 10% | How easily AI can access and parse content |

**Key Insight:** AEO is weighted equally to Content (35% each), signaling Searchable's strong belief that AI optimization is as important as traditional content quality.

## Updated Strategic Recommendations

Based on documentation findings, additional differentiation opportunities:

### 1. Transparent Scoring
- Publish your exact scoring methodology upfront
- Let users understand how to improve specific components
- Searchable documents this well - match or exceed their transparency

### 2. API Democratization
- Offer API access at Professional tier ($125 equivalent)
- Searchable gates API to Custom only
- This could be a significant competitive advantage for agencies

### 3. Extended Data Retention
- Offer 90-day retention on Starter tier
- 1-year on Professional
- Unlimited on Scale
- Beats Searchable's restrictive retention model

### 4. Platform Coverage Strategy
- Match current coverage: ChatGPT, Claude, Gemini, Perplexity
- Beat them to market on: Grok, DeepSeek, Meta Llama
- Add regional/specialized: Baidu, Yandex, regional AI assistants

### 5. Scoring Component Weights
Consider different weightings for your market:
- Increase Technical weight for enterprise clients (security, speed)
- Increase AEO weight for early adopters (AI-first businesses)
- Offer customizable weights per industry vertical
