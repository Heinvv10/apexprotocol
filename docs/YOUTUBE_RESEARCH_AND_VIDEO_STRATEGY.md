├─ Action: Show template on screen
└─ Explain: "These should be real customer questions"

Demo:
├─ Open ChatGPT → enter first query → screenshot
├─ Open Claude → enter query → screenshot
├─ Open Gemini → enter query → screenshot

└─ Narrate: "Notice if you get mentioned, your position, what competitors appear"

[STEP 3: SCORE YOUR RESULTS - 5:30-7:00]
Calculate:
├─ Count mentions across all queries
├─ Calculate: Mentions / Total queries × 100 = % visibility
├─ Compare to industry benchmarks
└─ Explain what score means (0-20% = poor, 50%+ = good)

[STEP 4: WHAT TO DO WITH THIS - 7:00-8:00]
Action items:
├─ "If <20% visible: You need content optimization"
├─ "If 20-50% visible: You're competitive but can improve"
├─ "If >50% visible: You're winning"
└─ Next steps: Use Apex tool for deeper analysis

[RESOURCES SHOWN]
✓ Query template (downloadable link in description)

✓ Next steps decision tree

✓ "Download the free template" (gated lead magnet)
✓ "Use Apex for automated audits" (free trial)
